<h2 id="개요"> 🎈 개요</h2>

 옛날부터 꿈꿔왔던 마을 재건 프로젝트를 위해, 공동체의 대표인 당신은 최대한 많은 자산을 모아야 합니다. 
마을은 오랜 시간 동안 황폐해졌고, 이제 주민들의 삶을 다시 풍요롭게 만들기 위해 노력해야 합니다.<br> 
 다양한 마을을 여행하며 작물을 사고팔고, 시장의 변동을 예측하여 가장 큰 이익을 얻어 보세요.<br> 
모은 자산은 마을을 재건하는 데 사용되며, 궁극적으로는 마을을 다시 번영하게 만들어 주민들의 행복을 되찾는 것이 목표입니다.<br> 
당신의 전략과 결단력으로 황폐한 마을을 아름답게 변화시켜 보세요! 💒✨

- 우리 게임은 플레이어가 다양한 마을에서 작물을 매수하고 매도하며 시세 변동을 이용해 최대한의 이익을 얻는 것을 목표로 하는 전략 시뮬레이션 게임입니다. 
- 친구들과 함께 게임방을 만들어 플레이하거나, 다른 플레이어와 경쟁하며 주간 랭킹에서 높은 순위를 차지할 수 있습니다. 
- 각 라운드마다 변화하는 시장 상황을 잘 파악하고 전략적으로 행동하여 최고의 자산을 쌓아보세요!

<h2 id="게임-기능-설명"> 🎲 게임 기능 설명 </h2>
<!-- 홍선주 -->

| 카테고리 | 기능 |
|:---:|:---|
| 회원 관련 기능 | - 소셜 로그인 (Kakao 로그인)<br/> - 최초 로그인 시 닉네임 설정<br/> - 회원 정보 수정 (닉네임, 한 줄 소개 수정) |
| 아바타 관련 기능 | - 아바타 목록 조회<br/> - 아바타 상세 조회<br/> - 아바타 구매<br/> - 아바타 장착 |
| 친구 관련 기능 | - 닉네임을 통한 사용자 검색<br/> - 친구 신청<br/> - 친구 신청 알림<br/> - 친구 신청 수락/거절<br/> - 친구 목록 조회<br/> - 친구 상세 조회 |
| 게임방 관련 기능 | - 게임방 생성(일반방 & 비밀방)<br/> - 비밀번호 입력 시 자동으로 비밀방으로 생성<br/> - 게임방 입장<br/> - 게임방 목록 조회<br/> - 생성된 방 중 랜덤으로 입장 가능 |
| 알림 기능 (실시간 알림) | - 알림 목록 조회<br/> - 친구 신청, 게임방 초대 알림 조회 |
| 주간 랭킹 | - 게임 종료 후 순위별로 지급되는 경험치의 합산으로 주간 랭킹 결정<br/> - 매주 월요일 자정 초기화 |
| 게임방 방장 | - 게임방 생성 시 자동 방장 부여<br/> - 방장이 나갈 시 다른 플레이어에게 방장 권한 부여<br/> - 모든 플레이어 준비 완료 시 게임 시작<br/> - 방장이 아닌 참여자는 준비 버튼 클릭 가능<br/> - 채팅 기능 |
| 게임방 친구 초대 | - 게임 대기방 내에서 온라인 상태인 친구 목록 조회<br/> - 게임방 친구 초대<br/> - 친구 초대 시 초대받은 친구에게 알림 전송<br/> - 비밀방 초대는 비밀번호 입력 없이 입장 가능 |
| 게임 진행 | - 라운드 시작 (각 라운드 당 2분)<br/> - 라운드 진행 중 채팅 가능<br/> - 방향키로 이동, 스페이스 바로 상호작용<br/> - [M]키 클릭 시 맵 보기 가능 |
| 작물 매수 및 매도 | - 마을 별로 테마와 마을 이름이 부여 (과일, 채소, 약용식물 등..)<br/> - 매 게임마다 마을별로 랜덤으로 3개의 작물이 나옴<br/> - 각 마을의 작물을 매수, 매도 가능 (단, 정해진 개수만큼만 가능)<br/> - 라운드 별로 기사 결과에 따라 가격 변동 |
| 작물 기사 거래소 | - 랜덤한 작물 기사 구매 가능<br/> - 해당 작물 기사에 따라 다음 라운드 작물의 가격 변동 |
| 라운드 종료 시 | - 5 라운드 종료 시 보유 작물 모두 게임 내 자산으로 변환<br/> - 최종 순위 및 총 자산, 지급 경험치 및 마일리지 출력 |

<h3> 🎯 목표 </h3>

- 게임이 끝날 때 까지 기사를 통해 시세를 분석하고, <br/> 작물 구매 및 판매를 통해 가장 많은 이득을 내야합니다.

<br/>

| 키보드 | ↑ | ↓ | ← | → |

<h3> 📢 라운드 진행 </h3>

| 총 라운드 | 각 라운드 진행시간 |
|:---:|:----------:|
|5라운드| 2 : 00 |

- 총 5라운드로 진행되며, 각 라운드는 2분으로 구성하였습니다.
- 각 플레이어는 2분간, 기사를 보고, 혹은 감으로 전략적으로 작물을 구매하여 시세 변동을 기다립니다.

<br/>

|:---:|:--------:|
| 키보드 | SpaceBar |

- 거래소에서는 마을에 대한 랜덤기사를 구매하여, 시세 변동을 예측할 수 있습니다.
